# Security Policy

## Supported Versions

As the project is currently in active development and does not have formal versioning yet, we only provide security support for the latest state of the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| Latest (main branch) | + |
| All older commits   | -      |

## Reporting a Vulnerability

If you discover a security bug or a critical vulnerability, please **do not open a public Issue**. 

Instead, please report the issue using:

1. **GitHub Private Reporting:** Use the [Private Vulnerability Reporting](https://github.com/avsbestua/iprad/security/advisories/new) feature under the "Security" tab.
2. **Email:** Send a report to: `widzmien333@gmail.com`.
